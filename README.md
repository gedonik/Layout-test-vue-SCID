
# Layout-test-vue-SCID is a Vue application.

Vue app test layout for SCID company.


## Installation

Clone my project, after type in terminal:

```bash
  npm install 
  npm run serve
```

## Tech Stack

HTML, CSS, SCSS, Vue, Swiper.
